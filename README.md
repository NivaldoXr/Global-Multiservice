# GlobalMultiservice Corporate Website
Professional website for a vehicle import business, featuring a custom admin panel and automated tools.

## 🚀 Features
- **Custom Admin Panel:** Drag-and-drop image management for non-technical staff.
- **Import Cost Calculator:** Interactive tool with WhatsApp integration.
- **Automated Quotes:** Automatic PDF generation for client estimates.
- **Cloud Integration:** Optimized image storage using Cloudinary.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend/Hosting:** Firebase Hosting
- **Storage:** Cloudinary
